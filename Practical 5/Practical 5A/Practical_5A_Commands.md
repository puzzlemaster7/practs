# Practical 5A Commands

## Sysinternals Setup

Download and extract Sysinternals Suite from the official Microsoft site, then open PowerShell in the `SysinternalsSuite` folder.

## Commands Shown In The Practical

```powershell
.\logonsessions.exe
.\pslist
openfiles /query
.\psfile.exe
```

## Notes

- `logonsessions.exe` lists logon session information.
- `pslist` shows running process information.
- `openfiles /query` checks for files opened remotely.
- `psfile.exe` lists files and directories opened remotely.
