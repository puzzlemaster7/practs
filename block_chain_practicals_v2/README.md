# Blockchain Practical (Course Code: 612)

This workspace contains a per-practical folder structure with:
- `code/` (source code)
- `output/` (screenshots / logs / sample runs)
- `README.md` (original question, practical details, explanation, install/run steps for Windows + Ubuntu)

Open `COURSE_INFO.md` for the full course outline and `PRACTICALS_INDEX.md` for a clickable list of all practicals.

## Folder naming
- Parent folders: `practical_01`, `practical_02`, ...
- Subfolders: `a_...`, `b_...` (each runs independently on a fresh system)
